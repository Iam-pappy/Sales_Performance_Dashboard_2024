# Sales_Performance_Dashboard_2024
 📊 Sales Performance Dashboard 2024

## Project Overview
Interactive Power BI dashboard analyzing **$19.7M in sales** across 4 regions, 8 product categories, and 8 salespeople. Built from 1,800+ transactions.

## 🎯 Key Business Questions Answered
- Which region generates highest revenue & profit?
- Who are top 5 salespeople?
- Which products have highest margins?
- What are monthly/quarterly trends?
- Where are losses occurring?

## 📈 Key Findings

| Metric | Value |
|--------|-------|
| **Total Revenue** | $19,706,458 |
| **Total Profit** | $4,967,093 |
| **Average Margin** | 25.21% |
| **Total Units Sold** | 18,391 |

### Top Performers
- **Region:** West ($5.35M revenue, $1.35M profit)
- **Salesperson:** Grace ($2.77M revenue)
- **Product:** Tablet ($2.9M revenue)
- **Category:** Electronics (51.8% of total)

### Seasonal Trends
- Peak Month: January ($1.77M)
- Best Quarter: Q1 ($4.99M)

## 🛠️ Tools Used
- **Excel** - Data cleaning, calculated columns, pivot tables
- **Power BI** - Interactive dashboard, DAX measures
- **DAX** - Time intelligence, rankings, custom metrics

## 📊 Dashboard Pages
1. **Executive Dashboard** - KPIs, regional breakdown, monthly trends
2. **Salesperson Analysis** - Rankings, efficiency matrix, profit buckets
3. **Product Performance** - Product matrix, decomposition tree, revenue distribution

## 📁 Files in Repository
| File | Description |
|------|-------------|
| `Sales_Dashboard.pbix` | Power BI dashboard file |
| `Sales_Dashboard_Cleaned.xlsx` | Cleaned dataset with calculated columns |
| `Executive_Summary.pdf` | Business insights and recommendations |
| `DAX_Measures.txt` | All DAX formulas used |
| `screenshots/` | Dashboard images |

## 🚀 How to Use
1. Download `Sales_Dashboard.pbix`
2. Open with Power BI Desktop (free)
3. Explore slicers: Quarter, Region,Prifit Bucket, Category
4. Review insights in `Executive_Summary.pdf`
